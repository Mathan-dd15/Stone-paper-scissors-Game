# Stone-Paper-Scissors Game

This is a simple implementation of the classic "Stone-Paper-Scissors" game using Python and Tkinter library for creating the graphical user interface.

## How to Play

1. Clone or download this repository to your local machine.
2. Make sure you have Python installed.
3. Navigate to the project directory and run the script `stone_paper_scissors.py`.
4. A window will appear with buttons for "Stone", "Paper", and "Scissors".
5. Click on your choice, and the computer will randomly select its choice.
6. The result will be displayed in a message box showing your choice, the computer's choice, and the result of the game.

## Requirements

- Python 3.x
- Tkinter (usually comes pre-installed with Python)

## Usage

To play the game, simply run the script `stone_paper_scissors.py` using Python.
